# Aryan Jaiswal – Portfolio

A modern, interactive portfolio built with Next.js, Tailwind CSS, and Framer Motion. Features multi-page architecture with dedicated sections for experience, projects, resume, and music covers.

[![Live Website](https://img.shields.io/badge/Visit-aryanjaiswal.in-blue?style=for-the-badge)](https://aryanjaiswal.in)
[![License: MIT](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)](LICENSE)

---

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Components**: Three.js, React Three Fiber
- **Deployment**: Netlify (auto-deploy on push to `main`)
- **Domain**: [aryanjaiswal.in](https://aryanjaiswal.in)

---

## Pages

| Route | Description |
|-------|-------------|
| `/` | Homepage with hero, highlights, experience summary |
| `/experience` | Detailed work experience timeline |
| `/projects` | Project showcase with live links and GitHub |
| `/resume` | Full resume with downloadable PDF |
| `/notesbyaryan` | Music covers and Instagram performances |

---

## Local Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Production build
npm run build
```

---

## Deployment

This project auto-deploys to Netlify on every push to `main`.
